# la

List all files

- Lists all files in the system with their IDs
- Shows a simplified view compared to `ls`

## Usage

```txt
username@system ~ % la
```

## Example

```txt
username@system ~ % la
ROOT (All) | Total Files: (125)
/Name

file1.txt                                    id 1
file2.txt                                    id 2
dir1/                                        id 3
...
```
