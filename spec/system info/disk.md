# disk

Display disk usage information

- Shows storage usage statistics

## Usage

```txt
username@system ~ % disk
```

## Example Output

```txt
username@system ~ % disk
Used        |   128MB
Total       |   1GB
Capacity    |   12.8%
Files       |   42
```

## Information Displayed

- Used storage space
- Total available storage space
- Capacity usage as a percentage
- Number of files stored
